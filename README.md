# Image_generation_app
# Приложение для генерации изображений

***

## Описание:
### Приложение представляет собой инструмент для генерации .png и .jpg файлов. Генерация происходит в два этапа. Изображения упрощенные

***

## Техническая справка:
### Как говорилось ранее, генерация происходит в два этапа. На первом модель обучается и создаёт набор данных в формате .json. в набор входит название категории, объекта, идентификатор объекта и SVG код. Второй этап преобразует SVG код в картинку. Такой подход был выбран для облегчения нагрузки на низкопроизводительные ПК без поддержки CUDA

***

## Назначения файлов:
### ● train_svg_model_simple.py - обучение модели
### ● svg_converter_simple.py - конвертер svg в картинку
### ● training_dataset.json - набор данных для тренировки нейросети. Можно добавить свои примеры
### ● best_model.pht - обученная модель
